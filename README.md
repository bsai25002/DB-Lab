# DB-Lab
**Course:** Database Security (DATABSECOURSE)  

## 📌 Short Intro

This repository contains my coursework, notes, and hands-on labs for the Database Security course. The focus is on understanding and implementing core security concepts such as authentication, access control, encryption, and secure database design. Through practical exercises and theoretical study, I explore how to protect sensitive data from unauthorized access and common threats like SQL injection and privilege abuse.

## 🧠 What I Am Learning

Data storage, B-trees, RDBMS, Flat File Strawman
